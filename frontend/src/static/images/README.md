# 图标资源说明

此目录用于存放项目所需的图标资源。

## 需要添加的图标文件

### TabBar 图标 (64x64px)
- `home.png` - 首页图标（未选中）
- `home-active.png` - 首页图标（选中）
- `chat.png` - 聊天图标（未选中）
- `chat-active.png` - 聊天图标（选中）
- `history.png` - 历史图标（未选中）
- `history-active.png` - 历史图标（选中）
- `profile.png` - 个人图标（未选中）
- `profile-active.png` - 个人图标（选中）

### 其他图标
- `logo.png` - 应用Logo
- `default-avatar.png` - 默认头像

## 图标规范

- 格式：PNG
- 大小：64x64px (TabBar图标)
- 背景：透明
- 风格：简洁、现代化医疗主题
- 颜色：
  - 未选中：#7A7E83
  - 选中：#1658FF

## 临时方案

项目当前使用emoji作为图标，如需使用实际图标文件，请将对应的PNG文件放入此目录。
